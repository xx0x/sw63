# SW63 Bootloader

The bootloader is a separate project created using TinyUF2, source available in [my fork](https://github.com/xx0x/tinyuf2/tree/sw63-bootloader) inside `ports/stm32l0/boards/sw63`.

You can enter the bootloader holding the Button and pressing the Reset. Or double-pressing the Reset if you prefer. Press the Reset again to restart back to the Application.
